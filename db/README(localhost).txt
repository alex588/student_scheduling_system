Only for Localhost

1. Download and Install XAMPP - (http://www.apachefriends.org/en/xampp-windows.html)
2. Launch XAMPP and make sure MYSQL and ApAche is checked and start them.
3. Add the mysql/bin file to your class path. It should be present where you installed XAMPP. In my case its at "G:\xampp\mysql\bin" and save it
4. Go to our project and navigate to the db file.
5. Double click db_schema.bat and db_init_values.bat (in the same order)
done
